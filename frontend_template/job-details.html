Senior Software Engineer
Tech Solutions Inc.  |  New York, NY
Full-time Remote $120k - $150k

Job Description

We are looking for a Senior Software Engineer to join our team. You will be responsible for designing, developing, and maintaining high-quality software solutions. The ideal candidate has experience with modern web technologies, cloud platforms, and agile methodologies.

  • Design and implement scalable software systems
  • Collaborate with cross-functional teams
  • Write clean, maintainable code
  • Participate in code reviews

Job Requirements

  • Bachelor's degree in Computer Science or related field
  • 3+ years of experience in web development
  • Proficiency in HTML, CSS, JavaScript, and React
  • Experience with version control systems (Git)
  • Strong problem-solving skills and attention to detail
  • Excellent communication and teamwork abilities
